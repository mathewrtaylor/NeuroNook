# NeuroNook (NN) GitHub Guides

# Abbreviated Novice Quick Guide (Markdown)

## What GitHub Is

GitHub is an organized online workspace where NN stores shared resources and documents.

You do not need technical experience to use it.

---

## Quick Setup

1. Create a GitHub account
2. Enable Two-Factor Authentication (2FA)
3. Accept your NN invite
4. Open repositories from the organization page

---

## Basic Navigation

### To View Files

1. Open the repository
2. Click folders to navigate
3. Click a file to preview it

---

## Requesting Changes

### Use Issues

1. Open the repository
2. Select **Issues**
3. Select **New Issue**
4. Describe the change needed

---

## Important Security Rules

Never upload:

- Personal information
- Passwords
- Sensitive conversations

Always keep your 2FA recovery codes safe.

---

## Helpful Reminders

- GitHub previews files automatically
- Search can help you find documents quickly
- Discussions are used for collaboration
- Issues are used for requests and problems
